# InstagramLikerGrabber
# A Python program to grab all users who have liked a post and append them to a spreadsheet of all previous posts this has been run on, sorting users by frequency of occurence
# Required installs, tkinter, selenium, pandas, xlwt
# NOTE: The path of all text files and the webdriver are hardcoded. This project is not permitted for commercial use however, if you intend to use this personally you must change # the paths to a suitable path in your machine.
# NOTE 2: This project is meant to work for accounts which have had no interaction with other posts I.e. Bots. You must add your bot username and password inside the code.
